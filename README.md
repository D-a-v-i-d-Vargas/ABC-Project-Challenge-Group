# ABC Project - Challenge Group

## 📝 Descripción del Proyecto
Este repositorio contiene el desarrollo del proyecto **"ABC"**, un sitio web interactivo y dinámico diseñado para mostrar el abecedario de forma visual utilizando elementos del entorno. El proyecto fue rescatado y completado bajo un enfoque de desarrollo ágil (Sprint rápido) para resolver problemas heredados de maquetación, estilos e interactividad.

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructuración del abecedario y corrección de errores semánticos.
* **CSS3:** Estilización responsiva, manejo de selectores específicos y paletas de colores dinámicas.
* **JavaScript:** Implementación de funciones lógicas para interactividad en tiempo real (cambios de color aleatorios).
* **Git & GitHub:** Gestión de ramas por tareas (`feature branches`) y organización del flujo de trabajo a través de GitHub Projects.

## 👥 Colaboradores
Desarrollado de manera conjunta por los miembros del departamento para asegurar el despliegue óptimo antes de la fecha límite.
