# ABC Project - Challenge Group 🚀

Proyecto web interactivo 'ABC' desarrollado en equipo mediante HTML, CSS y JavaScript. [cite_start]Enfoque en desarrollo front-end, resolución de bugs y flujos de trabajo colaborativos con Git/GitHub[cite: 4, 7, 8, 12].

---

## 📝 Descripción del Proyecto
[cite_start]Este repositorio contiene el desarrollo del proyecto **"ABC"**, un sitio web interactivo y dinámico diseñado para mostrar el abecedario de forma visual utilizando elementos del entorno[cite: 3, 7]. [cite_start]El proyecto fue rescatado y completado bajo un enfoque de desarrollo ágil (Sprint rápido) para resolver problemas heredados de maquetación, estilos e interactividad[cite: 4, 8, 9].

---

## 🛠️ Tecnologías Utilizadas
* [cite_start]**HTML5:** Estructuración del abecedario y corrección de errores semánticos[cite: 4, 20, 21, 22].
* [cite_start]**CSS3:** Estilización responsiva, manejo de selectores específicos y paletas de colores dinámicas[cite: 4, 23].
* [cite_start]**JavaScript:** Implementación de funciones lógicas para interactividad en tiempo real (cambios de color aleatorios)[cite: 4, 29, 31].
* [cite_start]**Git & GitHub:** Gestión de ramas por tareas (`feature branches`) y organización del flujo de trabajo a través de GitHub Projects[cite: 12, 15, 40].
